If you read us carefully, you’ve already seen that we use AngularJS for SubToMe.Angular is an open source web framework for the client side.It’s probably one of the most popular of its kind these days, so you’ve probably stumbled upon it a couple times.Today, we’re introducing a Superfeedr module for Angular.It implements all of our Web API so that it becomes trivial to build a client side application which consumes feeds.
